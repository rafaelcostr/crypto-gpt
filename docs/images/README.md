# Screenshots do README

Imagens usadas na seção **Demonstração** do [README](../../README.md).

## Regenerar

Com o app rodando (`npm run dev`):

```bash
npm run screenshots
```

O script `scripts/capture-screenshots.mjs` grava:

| Arquivo | Tela |
|---------|------|
| `demo-01-chat.png` | Chat com conversa de exemplo |
| `demo-02-portfolio.png` | Portfólio com BTC/ETH |
| `demo-03-news.png` | Notícias (RSS) |
| `demo-04-alerts.png` | Alertas de preço |

Para capturar com dados reais da IA, tire prints manualmente após testar com `GROQ_API_KEY` válida.
